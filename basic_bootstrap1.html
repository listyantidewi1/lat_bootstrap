Bootstrap Container

Ini adalah fixed container

Ini adalah container fluid

Container fluid akan memakan tempat 100% lebar halaman, tidak peduli berapapun lebar halamannya (layar laptop, layar tablet, layar HP, tetap memakan 100%)

bg-danger = memberi background warna merah
