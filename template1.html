Contoh Halaman Web Menggunakan Bootstrap

container-fluid = membuat kotak container yang responsif terhadap ukuran layar

p-5 => p=padding => p-5 artinya memberi padding sebesar 5

bg-primary = memberi background warna biru

text-white = memberi warna putih pada teks, text-center = memberi posisi tengah pada teks

Kolom 1

"container digunakan utk membuat container, mt-5 artinya memberi margin-top sebesar 5"

"row" digunakn utk membuat baris, yang dapat diisi elemen-elemen, misal kolom ini

Kolom 2

"col-sm-4" artinya membuat kolom dengan ukuran span sebesar 4 span

"col-sm-4" dapat dipelajari lebih lanjut nanti di Grid Layout

Kolom 3

Stay tuned untuk fitur bootstrap yg lain
